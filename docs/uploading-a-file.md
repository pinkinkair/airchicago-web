# Uploading a File

In general, you'll need to upload images and other simple files to the src/assets/ folder.

## How to Upload

1. Go to the [src/assets/ folder](https://github.com/pinkinkair/airchicago-web/tree/main/src/assets)
2. Make sure you're on the `main` branch. You'll see a dropdown above the Files list in the left sidebar that should say `main`. If it says anything else, switch to `main` in the dropdown.
3. Now on the right hand side of the page, you should see an Add File dropdown.
4. Click the dropdown and select Upload Files.
5. You should see a screen that says `Drag files here to add them to your repository` `or choose your files`. Either drag a file over or click "choose your files"
6. Once you've selected all your files, at the bottom of the screen, you'll see a `Create a new branch...` option. Use this textbox to name your branch, not using any spaces. For example, `images/logo-v4-and-background-green`.
  - Read more about branches in [our using branches docs](./using-branches-and-pull-requests.md#whats-a-branch)
7. Click the `Propose changes` button skip to the [create a Pull Request document](./using-branches-and-pull-requests.md#using-a-pull-request)
