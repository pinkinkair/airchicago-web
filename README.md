A simple GitHub Pages site. Deployment config for GitHub Pages is in .github/workflows/static-deploy.yml

## Development

### First time setup

> [!NOTE]
> If you are developing locally, you'll need Node and NPM installed

- Run `npm i` to install required dependencies

### General workflow

- Boot up a local instance of the site by using `npm run preview`
- Ensure the build succeeds before pushing to `main` by using `npm run build`
